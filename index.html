IClassPro Widget

A lightweight widget for embedding IClassPro class schedules on any website.

Let's Test It!

Quick Start

Add the widget script to your HTML:

<script 
    src="https://icp-widget.vercel.app/icp-widget.js"
    data-account-name="gymagination"
    data-levels="4"
    data-programs="56"
    data-days="2"
    async
    crossorigin="anonymous">
</script>

Required Parameters

  • data-account-name: Your IClassPro account name (e.g., "gymagination")

Optional Parameters

  • data-levels: Level filter (default: "4")
  • data-programs: Program filter (default: "56")
  • data-days: Days filter (default: "2")
  • data-container-id: Custom ID for the widget container (default: "icp-widget")

Live Demo

Note: The widget will automatically create its container if one doesn't exist. You can also pre-create a container with a specific ID: 
<div id="icp-widget"></div>

Styling

The widget inherits your website's fonts and colors. You can also style it using CSS:

#icp-widget {
    max-width: 800px;
    margin: 0 auto;
}

#icp-widget .class-card {
    border: 1px solid #eee;
    margin-bottom: 10px;
    padding: 15px;
}
